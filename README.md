# RingLite

A virtual ring light for video recording. Invisible to screen recorders.

![RingLite](screenshot.jpg)

## Features

- Resizable glowing white ring
- Drag to reposition
- **Invisible to screen recording** (macOS & Windows)
- Remembers size between sessions

## Controls

| Action | Control |
|--------|---------|
| Move | Drag |
| Resize | Scroll or `+`/`-` |
| Nudge | Arrow keys |
| Help | `H` |
| Quit | `Esc` |
